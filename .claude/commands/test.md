# Test Command

Run the appropriate tests for: $ARGUMENTS (or all tests if no argument provided)

## Steps:
1. Identify the test framework (Jest, rspec, pytest, go test, etc.)
2. Locate test files related to the target
3. Run tests with appropriate commands
4. If failures occur:
   - Analyze the error messages
   - Identify root cause
   - Suggest fixes
   - Re-run tests to verify
5. Report final status with pass/fail counts
